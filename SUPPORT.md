## ğŸ› Reporting Bugs

Good bug reports help us fix issues faster. Please include:

1. **Clear title**: Describe the issue concisely
2. **Environment details**: OS, version, relevant system info
3. **Steps to reproduce**: How to trigger the bug
4. **Expected behavior**: What should happen
5. **Actual behavior**: What actually happens
6. **Logs/screenshots**: Any helpful error messages or visuals

### Bug Report Template

```markdown
**Environment**
- OS: [e.g., Ubuntu 24.04]
- Version: [e.g., v0.7.0]
- Related: [any relevant details]

**Steps to Reproduce**
1. Go to '...'
2. Click on '....'
3. See error

**Expected Behavior**
A clear description of what you expected to happen.

**Actual Behavior**
What actually happened.

**Logs/Screenshots**
[Paste relevant logs or add screenshots]
```
